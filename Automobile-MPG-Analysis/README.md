# Automobile Engine Impact Analysis on MPG

## Overview
Analyzed how different engine and vehicle factors affect fuel efficiency (Miles Per Gallon) across multiple automobile models.

## Objective
Identify which engine characteristics have the strongest impact on MPG and provide data-driven recommendations to improve efficiency.

## Tools Used
R, Python, Tableau, Excel

## Key Insights
- Regression analysis revealed strong correlations between engine displacement, horsepower, and MPG.
- Visualizations highlighted efficiency differences across vehicle categories.
- Identified key variables that manufacturers can optimize to improve fuel economy.

## Results
Provided insights that could improve vehicle fuel efficiency by up to 10% based on model predictions.
