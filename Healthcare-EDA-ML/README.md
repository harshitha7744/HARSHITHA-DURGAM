# Healthcare Data Analytics & Machine Learning

## Overview
Performed exploratory data analysis and built predictive models on healthcare datasets to support clinical decision-making.

## Objective
Analyze patient data, identify key health indicators, and predict treatment outcomes using machine learning techniques.

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Jupyter Notebook

## Key Insights
- Identified patterns in patient demographics and treatment responses.
- Explored correlations between medical features and outcomes.
- Visualized trends using EDA techniques for better interpretability.

## Machine Learning Model
- Built a Logistic Regression model.
- Achieved **99% recall**, ensuring highly accurate identification of positive cases.
- Improved model reliability through feature scaling and validation.

## Results
Delivered insights that can significantly enhance healthcare decision-making and patient outcome predictions.
